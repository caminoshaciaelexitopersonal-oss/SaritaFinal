# EMERGENCY FLAGS DOC - SARITA

**Fecha:** 24 de Mayo de 2024
**Estado:** Sistema de Intervención Listo

## 1. FLAGS DE SOBERANÍA
| Flag | Impacto | Estado Base |
| :--- | :--- | :--- |
| **Operaciones Comerciales** | Bloquea emisión de facturas y cierres de caja. | ACTIVE |
| **Registro de Usuarios** | Detiene el onboarding de nuevos prestadores. | ACTIVE |
| **Agentes Inteligentes** | Congela toda la jerarquía SARITA. | ACTIVE |

## 2. MODO EMERGENCIA (KILL MODE)
- **Congelamiento Sistémico:** Un interruptor maestro que activa todos los bloqueos soberanos simultáneamente.
- **Acceso Administrativo:** Durante el congelamiento, el SuperAdmin mantiene acceso total para investigar y resolver la causa de la emergencia.
- **Notificación:** El sistema expone un banner global indicando el estado de intervención.

## 3. PROTOCOLO DE RECUPERACIÓN
La desactivación de los flags requiere una justificación que queda registrada permanentemente en el Bitácora de Soberanía.
