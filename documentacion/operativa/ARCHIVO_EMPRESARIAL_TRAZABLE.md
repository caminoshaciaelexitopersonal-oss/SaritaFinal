# ARCHIVO EMPRESARIAL TRAZABLE - SARITA

**Fecha:** 24 de Mayo de 2024
**Estado:** Certificado (Trazabilidad Legal)

## 1. GESTIÓN DOCUMENTAL SOBERANA
El sistema implementa una capa de Gestión Archivística que blinda la evidencia operativa de la empresa:

- **Subida de Documentos:** Interfaz para cargar contratos, facturas de compra, certificados SST y evidencias de prestación de servicios.
- **Asociación de Procesos:** Cada documento puede vincularse a una `Oportunidad Comercial`, una `Tarea Operativa` o un `Empleado`.

## 2. INTEGRIDAD Y SEGURIDAD
- **Hashing SHA-256:** Cada archivo subido genera un hash único registrado en la base de datos para garantizar que el contenido no ha sido alterado.
- **Sello de Tiempo:** Registro inmutable de la fecha y hora de carga y del usuario responsable.
- **Estado de Integridad:** Visualización en UI del check de validación del hash contra el registro original.

## 3. CUMPLIMIENTO REGULATORIO
- El módulo está diseñado para cumplir con los requisitos de la autoridad gubernamental (Vía 1), permitiendo auditorías rápidas sobre la documentación legal obligatoria de los prestadores.

## 4. CONCLUSIÓN
La Gestión Archivística no es un simple repositorio de archivos, sino un **sistema de custodia de integridad**. Asegura que la empresa Sarita tenga una base legal sólida y auditable, lista para inspecciones normativas.
