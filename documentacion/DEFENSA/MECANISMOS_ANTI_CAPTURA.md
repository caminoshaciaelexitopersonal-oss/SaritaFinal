# MECANISMOS ANTI-CAPTURA INSTITUCIONAL
**Salvaguardas contra la Concentración de Poder**

## 1. DETECCIÓN DE ANOMALÍAS SOBERANAS
El sistema monitorea el uso de las funciones de "Soberanía". Si un solo usuario ejecuta más del 80% de las intervenciones manuales en un nodo, se dispara una alerta de **Riesgo de Captura**.

## 2. SEGMENTACIÓN JURISDICCIONAL
Al ser un sistema de nodos federados, es imposible que el control de un municipio permita capturar el nodo nacional. La soberanía está distribuida, evitando puntos únicos de fallo institucional.

## 3. AUDITORÍA CRUZADA
Los auditores de un nivel superior (ej: Departamental) pueden revisar las políticas de un nivel inferior (ej: Municipal) para detectar sesgos o privilegios indebidos en las reglas de negocio cargadas en el Kernel.
