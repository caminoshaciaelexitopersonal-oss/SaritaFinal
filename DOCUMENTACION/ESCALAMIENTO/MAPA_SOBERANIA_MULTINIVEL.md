# MAPA DE SOBERANÍA MULTINIVEL
**Jerarquía de Decisión y Resolución de Conflictos**

## 1. NIVELES DE AUTORIDAD
| Nivel | Ámbito | Tipo de Decisión | Precedencia |
| :--- | :--- | :--- | :--- |
| **Supremo** | Constitución/Global | Derechos, Límites de IA, Bloqueos de Seguridad. | 1 (Máxima) |
| **Nacional** | País | Leyes de Turismo, Impuestos (DIAN), Auditoría. | 2 |
| **Regional** | Departamento | Promoción de Marca Región, Planes Viales. | 3 |
| **Local** | Municipio | Decretos Locales, Permisos de Funcionamiento. | 4 |

## 2. RESOLUCIÓN DE CONFLICTOS
- **Regla de Oro:** Un nivel inferior nunca puede anular una política de un nivel superior.
- **Escalamiento:** Si un `Teniente` de IA detecta una regla local que contradice una nacional, la acción se bloquea y se genera una **Alerta de Inconsistencia Normativa** para el SuperAdmin Nacional.

## 3. TRAZABILIDAD MULTI-NODO
El sistema registra el "Path de Autoridad" de cada decisión: `[GLOBAL] -> [NACIONAL] -> [REGIONAL] -> [LOCAL]`.
