# CADENA DE MANDO IA REAL

## 1. JERARQUÍA OPERACIONAL
- **Strategic Agents (Level 6):** Toman decisiones de expansión y riesgo sistémico.
- **Governance Agents (Level 5):** Auditan el cumplimiento de RLS y leyes territoriales.
- **Orchestrator Agents (Level 4):** Distribuyen tareas a los workers tácticos.
- **Tactical Agents (Level 1-3):** Ejecutan tareas específicas (Facturación, Reservas).

## 2. INTERACTION FLOW
Los agentes se comunican de forma asíncrona mediante el Bus de Eventos, siguiendo protocolos de "Misión" y "Reporte de Resultado".
