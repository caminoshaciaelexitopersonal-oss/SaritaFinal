# Key Management Inventory (Phase 84.1)

| Asset | Type | Lifecycle Status | Risk |
| :--- | :--- | :--- | :--- |
| **Root Secret** | Anchor | Permanent (Static) | Critical |
| **Court Signature** | Authority | Non-Expiring | High |
| **Authority Tokens** | Functional | Non-Expiring | Medium |
| **Component Hashes** | Identity | Static | Low |

## Targeted Lifecycle States
1. **Creation:** Intent-based generation.
2. **Activation:** Formal registration in Trust Ledger.
3. **Rotation:** Scheduled replacement without service interruption.
4. **Revocation:** Emergency invalidation.
5. **Retirement:** Controlled sunsetting of old anchors.
