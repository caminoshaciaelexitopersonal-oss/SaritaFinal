# Logical Consistency Gap Matrix

| Component | Logical Gap | Risk | Priority |
|---|---|---|---|
| Inference | No Modus Ponens/Tollens | Reasoning is linear and non-branching. | High |
| Expressions | No symbolic logic parser | Reliance on fragile string matching. | High |
| Consistency | No contradiction detection | Mutually exclusive constraints can co-exist. | High |
| Axioms | No foundational truths | Proofs are "floating" without grounding. | Medium |
| Derivation | Static proof steps | Conclusions are asserted, not derived. | High |
| Completeness | Limited proof space | Cannot prove complex multi-step transformations. | Medium |
