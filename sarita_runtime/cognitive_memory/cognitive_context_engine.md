# MOTOR DE CONTEXTO COGNITIVO

## 1. RECONSTRUCCIÓN CONTEXTUAL
Antes de ejecutar una tarea, el motor de contexto une:
- Datos relacionales (SQL).
- Recuerdos semánticos (Vectores).
- Políticas vigentes (Infrastructure).

## 2. REASONING CONTEXT
El "Context Window" del LLM se alimenta de forma dinámica, priorizando la información más reciente y relevante para la tarea actual, evitando la saturación cognitiva del agente.
