# Catálogo de Comandos de Voz - Super Administrador (SITYC)

Este catálogo define las intenciones y ejemplos de comandos verbales soportados por SARITA para la operación del ERP administrativo.

## 1. Dominio Comercial
- **Intención**: `ERP_VIEW_SALES_STATS`
  - *Ejemplos*:
    - "Sarita, ¿cuáles son las estadísticas de ventas de este mes?"
    - "Dime cuánto hemos vendido en suscripciones esta semana."
- **Intención**: `ADMIN_VENTAS`
  - *Ejemplos*:
    - "Sarita, registra una nueva venta de plan premium."

## 2. Dominio Contable
- **Intención**: `ERP_CREATE_VOUCHER`
  - *Ejemplos*:
    - "Sarita, crea un comprobante contable por un millón de pesos por servicios de limpieza."
- **Intención**: `ERP_GENERATE_BALANCE`
  - *Ejemplos*:
    - "Genera el balance general del trimestre actual."

## 3. Dominio Financiero
- **Intención**: `ERP_VIEW_CASH_FLOW`
  - *Ejemplos*:
    - "Sarita, muéstrame el flujo de caja proyectado para el próximo mes."
- **Intención**: `ADMIN_FINANCIERO`
  - *Ejemplos*:
    - "Registra un ingreso de tesorería por renovación de licencias."

## 4. Dominio Operativo
- **Intención**: `ERP_MANAGE_RESOURCES`
  - *Ejemplos*:
    - "Sarita, asigna más almacenamiento al servidor de archivística."
    - "Aumenta la prioridad de procesamiento para el onboarding."

## 5. Dominio Archivístico
- **Intención**: `ERP_SEARCH_DOCUMENT`
  - *Ejemplos*:
    - "Sarita, busca el contrato del prestador 'Hotel Sol'."
    - "Encuentra la resolución administrativa de la semana pasada."

## 6. Auditoría y Seguridad
- **Intención**: `AUDITORIA_GLOBAL`
  - *Ejemplos*:
    - "Sarita, ejecuta una auditoría de seguridad inmediata."
- **Intención**: `PLATFORM_SUSPEND_USER`
  - *Ejemplos*:
    - "Suspende al usuario Juan Pérez por violación de términos."
