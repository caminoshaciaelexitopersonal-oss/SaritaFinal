# Manual de Operación por Voz (SADI)

## 1. Introducción

Bienvenido a SADI (Sistema de Acceso y Despliegue Inteligente), su asistente de voz para la administración de la plataforma Sarita. Este manual proporciona una guía sobre cómo utilizar comandos de voz para realizar tareas administrativas de manera rápida y eficiente.

## 2. ¿Cómo Funciona?

Para usar SADI, simplemente active la interfaz de voz en su panel de administración y diga su comando en lenguaje natural. SADI interpretará su solicitud, la procesará y le dará una respuesta audible.

**Nota de Seguridad:** Para acciones que modifican datos importantes (como crear un plan) o que son irreversibles (como desactivar un módulo), SADI le pedirá una confirmación verbal antes de proceder.

## 3. Catálogo de Comandos Disponibles

A continuación se muestra una lista de los comandos que SADI entiende actualmente. Puede formular sus frases de manera natural; los ejemplos son solo una guía.

### Módulo: Gestión de Planes y Suscripciones

| Para... | Diga algo como... | Ejemplo |
| :--- | :--- | :--- |
| **Crear un nuevo plan** | "Crear un nuevo plan llamado..." | `"Sarita, crea un nuevo plan llamado Básico con precio 19.99 MENSUAL."` |
| **Desactivar un plan** | "Desactivar el plan con ID..." | `"Desactiva el plan 5."` (requiere confirmación) |
| **Cambiar el precio** | "Cambiar el precio del plan..." | `"Cambia el precio del plan Básico a 25.50."` |
| **Listar planes** | "Muéstrame los planes activos" | `"¿Cuáles son los planes que están activos actualmente?"` |

### Módulo: Gestión de Contenidos

| Para... | Diga algo como... | Ejemplo |
| :--- | :--- | :--- |
| **Publicar una página** | "Publicar la página..." | `"Publica la página de 'Contacto'."` |
| **Ocultar una página** | "Ocultar la página..." | `"Oculta la página de 'Términos y Condiciones'."` |

### Módulo: Gestión de Publicaciones (Blog, Noticias)

| Para... | Diga algo como... | Ejemplo |
| :--- | :--- | :--- |
| **Aprobar una publicación** | "Aprobar la publicación..." | `"Aprueba la publicación número 123."` |
| **Rechazar una publicación** | "Rechazar la publicación..." | `"Rechaza la publicación 123."` |
| **Listar pendientes** | "Muéstrame las publicaciones pendientes" | `"¿Hay alguna publicación pendiente de revisión?"` |
