# Arquitectura General SARITA v1.0
**Principio:** Omnicanalidad Soberana.

## 1. Capas del Sistema
1. **Interfaces:** Web (Next.js), Mobile (Expo), Desktop (Electron).
2. **Abstracción:** @sarita/shared-ui (Interfaz) y @sarita/shared-sdk (Lógica).
3. **Núcleo:** Backend Django 5.0 (REST API v1).
4. **Inteligencia:** Jerarquía militar N1-N7 para autonomía operativa.
5. **Persistencia:** PostgreSQL 15 + Redis 7 + SQLite (Offline).
