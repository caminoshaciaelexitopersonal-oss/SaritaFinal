# MÓDULO: POS (PUNTO DE VENTA)
**Función:** Procesamiento de transacciones y ventas locales.

## Características por Plataforma
- **Web:** Terminal de ventas ligera para navegadores.
- **Mobile:** POS móvil con integración de carrito y pagos rápidos.
- **Desktop:** POS completo con soporte para impresoras térmicas y escáneres.

## Componentes Shared-UI Usados
- `CartScreen`, `POSInterface`, `PaymentGateway`.
