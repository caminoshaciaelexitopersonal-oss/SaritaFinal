# MÓDULO: ERP (GESTIÓN EMPRESARIAL)
**Función:** Sistema administrativo integral para el prestador de servicios.

## Características por Plataforma
- **Web:** Gestión profunda de nómina, contratos y archivo legal.
- **Mobile:** ERP simplificado para registro de novedades y consulta de documentos.
- **Desktop:** ERP completo con operación offline y sincronización diferida.

## Componentes Shared-UI Usados
- `EmployeeManager`, `PayrollDashboard`, `ArchiveExplorer`, `SyncEngine`.
