# MAPA MAESTRO DEL SISTEMA SARITA v1.0

## 1. El Sistema de Sistemas
SARITA no es una aplicación única, es un ecosistema orquestado que conecta tres actores principales mediante una red de datos soberana.

### Vía 1: Gobernanza (Admin Control Tower)
- **Propósito**: Supervisión regional, analítica territorial y políticas públicas.
- **Módulos**: SADI, Control Tower, Auditoría Forense.

### Vía 2: Productividad (Mi Negocio / ERP)
- **Propósito**: Empoderamiento del prestador de servicios y artesanos.
- **Módulos**: Contabilidad, Inventarios, Nómina, Gestión de Reservas.

### Vía 3: Experiencia (Descubre Turismo / Marketplace)
- **Propósito**: Conexión directa entre el turista y la oferta local.
- **Módulos**: Catálogo Universal, Recomendaciones IA, Motor de Pagos.

## 2. Flujo de Datos Inteligente
1. **Captura**: El turista busca en la Web/Mobile (Marketplace).
2. **Predicción**: La IA analiza la demanda (Intelligence) y sugiere precios dinámicos.
3. **Transacción**: Se procesa el pago vía Wallet y se registra en el Ledger inmutable.
4. **Operación**: El prestador recibe la orden en su Dashboard y coordina con Delivery.
5. **Gobernanza**: El sistema SADI extrae el KPI de impacto económico para el Dashboard gubernamental.

## 3. Capas de Integridad
- **Blockchain Ready**: Hash SHA-256 encadenado en cada movimiento financiero.
- **Agent Governance**: Los agentes jerárquicos (N1-N7) ejecutan la estrategia sin intervención humana para procesos de bajo riesgo.
- **Zero-Touch**: Despliegue automático de nuevos inquilinos en la infraestructura K8s.
