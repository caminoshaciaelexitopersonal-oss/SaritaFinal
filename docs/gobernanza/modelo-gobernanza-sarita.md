# Modelo de Gobernanza Ecosistema SARITA v1.0
**Principio:** Un Cerebro, Muchos Cuerpos - Bajo Control Soberano.

## 1. Arquitectura de Gobierno
El sistema opera bajo un `GovernanceKernel` centralizado que supervisa:
- Cumplimiento de Políticas Institucionales.
- Salud Sistémica (Infraestructura + IA).
- Trazabilidad de Acciones Administrativas.

## 2. Jerarquía de Control
1. **Super Administrador:** Control total de infraestructura y políticas.
2. **Administración Institucional:** Supervisión regional y validación de prestadores.
3. **Prestadores:** Gestión empresarial autónoma bajo marco institucional.
4. **Usuarios:** Consumo de servicios con identidad soberana.

## 3. Gobierno de Datos
- **Clasificación:** Tiered storage (SENSITIVE, CRITICAL, OPERATIONAL).
- **Acceso:** RBAC + ABAC con validación de propiedad de recursos.
- **Inmutabilidad:** Logs de auditoría con Chained Hash (SHA-256).
