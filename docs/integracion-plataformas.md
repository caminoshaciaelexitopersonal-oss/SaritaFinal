# Integración de Plataformas
SARITA opera como una única plataforma distribuida:
- **Web:** Centro administrativo y de configuración.
- **Mobile:** Gestión operativa en campo y experiencia del turista.
- **Desktop:** Terminal de alto tráfico (POS) y resiliencia total offline.

Toda la comunicación se realiza vía **JWT RS256** sobre la API **v1**.
