# LÍMITES DE AGENTES EN CONTEXTO SUPRANACIONAL (SUPRA AGENTS LIMITS)

**Versión:** 1.0 (Fase Z-SUPRA)
**Estado:** ESTRATÉGICO / CRÍTICO
**Principio:** La IA observa y analiza; nunca ejecuta ni obliga.

---

## 1. TIPOS DE AGENTES PERMITIDOS
En el despliegue supranacional de SARITA, solo se autorizan los siguientes perfiles de "Funcionarios Digitales":

### 1.1 Agente Observador
*   **Misión:** Monitorear indicadores agregados de múltiples nodos para detectar tendencias regionales.
*   **Capacidad:** Solo lectura de eventos autorizados (Nivel 0 y 1).

### 1.2 Agente Verificador
*   **Misión:** Validar que los datos reportados por un Nodo Nacional coincidan con las reglas matemáticas del tratado.
*   **Capacidad:** Generar reportes de consistencia técnica.

### 1.3 Agente Auditor
*   **Misión:** Supervisar la integridad de los logs diplomáticos y la cadena de hashes del Supranational Kernel.
*   **Capacidad:** Certificar que el sistema no ha sido manipulado por ninguna de las partes.

### 1.4 Agente Mediador Técnico
*   **Misión:** Proponer soluciones matemáticas o ajustes técnicos ante una controversia de datos entre dos Estados.
*   **Capacidad:** Simulación de escenarios de acuerdo basados en parámetros históricos.

## 2. PROHIBICIONES ABSOLUTAS (RED LINES)

Queda terminantemente prohibido el despliegue de:

1.  **❌ Agentes Ejecutores:** Ninguna IA supranacional puede realizar cambios en la configuración operativa de un país.
2.  **❌ Agentes Coercitivos:** El sistema no tiene capacidad para imponer sanciones, bloqueos económicos o acciones físicas automáticamente.
3.  **❌ Agentes de Infiltración:** Está prohibido el uso de agentes para extraer datos no autorizados de los Nodos Nacionales (NC).
4.  **❌ Agentes de Decisión Soberana:** La IA nunca puede sustituir el voto o el consenso de los representantes humanos de los Estados.

## 3. GOBERNANZA DE AGENTES SUPRA
*   **Auditoría de Algoritmos:** Todas las partes firmantes tienen derecho a revisar el código y los "prompts" de los agentes supranacionales para asegurar neutralidad.
*   **Kill Switch de Agente:** Cualquier Estado puede solicitar la desactivación inmediata de un agente supranacional si considera que está violando su soberanía o los términos del tratado.
